1. Matplotlib

1.1 Plotting

By default, the `plot()` function draws a line from point to point. The function takes parameters for specifying points in the diagram. Parameter 1 is an array containing the points on the x-axis. Parameter 2 is an array containing the points on the y-axis.


import numpy as np
import matplotlib.pyplot as plt
x = np.array([1, 8])
y = np.array([3, 10])

plt.plot(x, y)
plt.show()

# Without Line
plt.plot(x, y, 'o')
plt.show()

# Default X-Points
plt.plot(y)
plt.show()

1.2 Markers

We can use the keyword argument `marker` to emphasize each point with a specified marker


y = np.array([3, 8, 1, 10])

plt.plot(y, marker = 'o')
plt.show()

plt.plot(y, marker = '*')
plt.show()

1.2.1 Format Strings `fmt`

We can also use the *shortcut string notation* parameter to specify the marker. This parameter is also called `fmt`, and is written with this syntax: *marker*|*line*|*colo


y = np.array([3, 8, 1, 10])

plt.plot(y, 'o:r')
plt.show()

1.2.2 Marker Size

We can use the keyword argument `markersize` or the shorter version, `ms` to set the size of the markers:


y = np.array([3, 8, 1, 10])

plt.plot(y, marker = 'o', ms = 20)
plt.show()

1.2.3 Marker Color

We can use the keyword argument `markersize` or the shorter version, `ms` to set the size of the markers:


y = np.array([3, 8, 1, 10])

plt.plot(y, marker = 'o', ms = 20, mec = 'r')
plt.show()

We can use the keyword argument `markerfacecolor` or the shorter `mfc` to set the color inside the edge of the markers


y = np.array([3, 8, 1, 10])

plt.plot(y, marker = 'o', ms = 20, mfc = 'r')
plt.show()

4. Seaborn Library

Seaborn is a powerful Python library that makes it easy to create informative and attractive data visualizations.

4.1 Box Plot()

A `box plot` helps to maintain the distribution of quantitative data. The main body of the box plot showing the quartiles and the median's confidence intervals if enabled. The medians have horizontal lines at the median of each box and while whiskers have the vertical lines extending to the most extreme, non-outlier data points and caps are the horizontal lines at the ends of the whiskers.


import matplotlib.pyplot as plt
import seaborn as sns


sns.set(style='whitegrid')
# Load dataset
tip = sns.load_dataset('tips')  

# Create a figure and axis for better control (optional but recommended)
plt.figure(figsize=(12, 8))

# Boxplot of total_bill by day, colored by smoker
sns.boxplot(x="day", y="total_bill", hue="smoker",
            data=tip, palette="Set2", dodge=True)  # Fixed: 'tips' → 'tip'

# Labels and title
plt.ylabel("Tip ($)", size=14)
plt.xlabel("Day", size=14)
plt.title("Tips Analysis", size=18)

# Adjust layout to prevent overlap
plt.tight_layout()

# Show plot
plt.show()

4.2 Violin Plot()

A `violin plot` plays a similar activity that is pursued through `whisker or box` plot do


import matplotlib.pyplot as plt
import seaborn as sns

sns.set(style='whitegrid')
fmri = sns.load_dataset("fmri")
plt.figure(figsize=(12, 6))

sns.violinplot(
    x="timepoint",
    y="signal",
    hue="region",          # two levels → works with split=True
    data=fmri,
    palette="Set2",
    split=True,            # one half per region
    scale="count",         # width proportional to observation count
    linewidth=1.5,
    inner="quartile"       # optional: show quartiles inside
)

plt.title("FMRI Signal – Split by Region (with event-styled points)", size=16)
plt.xlabel("Timepoint", size=12)
plt.ylabel("Signal", size=12)

plt.tight_layout()
plt.show()

4.3 Swarmplot()

Draw a `categorical scatterplot` with non-overlapping points. This function is similar to `stripplot`


import matplotlib.pyplot as plt
import seaborn as sns

# Load dataset
data = sns.load_dataset("tips")

plt.figure(figsize=(10, 6))

sns.swarmplot(
    y="day",           # ← categorical on y-axis for horizontal
    x="total_bill",    # ← numeric on x-axis
    hue="size",        # continuous → color by size
    data=data,
    orient="h",        # horizontal
    size=5,
    palette="viridis", # better for continuous
    dodge=False        # ← MUST be False for continuous hue
)

plt.title("Total Bill by Day (Horizontal Swarmplot)", size=16)
plt.xlabel("Total Bill ($)", size=12)
plt.ylabel("Day", size=12)

# Improve legend
plt.legend(title="Party Size", bbox_to_anchor=(1.05, 1), loc='upper left')

plt.tight_layout()
plt.show()

4.7 Correlation Heatmap

A correlation heatmap is a heatmap that shows a 2D correlation matrix between two discrete dimensions, using colored cells to represent data from usually a monochromatic scale.


# Import necessary libraries
import pandas as pd
import seaborn as sns
import matplotlib.pyplot as plt

# Load a sample dataset (tips dataset from seaborn)
df = sns.load_dataset('tips')

# Select only numeric columns for correlation
numeric_df = df.select_dtypes(include=['float64', 'int64'])

# Compute the correlation matrix
corr_matrix = numeric_df.corr()

# Create a heatmap
plt.figure(figsize=(10, 6))
sns.heatmap(corr_matrix, annot=True, cmap='coolwarm', center=0, linewidths=0.5)
plt.title('Correlation Heatmap of Tips Dataset')

plt.tight_layout()
plt.show()
