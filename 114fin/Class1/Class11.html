List

A list is an ordered collection of items which can be of varied data types. Lists are mutable, meaning they can be altered after their creation. A list is a built-in data type used to store an ordered collection of items. These items can be of different types (e.g., integers, strings, other lists, etc.), and lists are **mutable** , meaning their contents can be changed after creation.

Tuples

A tuple is an ordered, immutable collection of items. Like lists, tuples can store multiple items in a single variable, but unlike lists, tuples cannot be changed after creation (i.e., they are immutable)

Creating a Tuple

Similar to a list in terms of indexing, nested objects, and iteration. However, unlike lists, tuples are immutable. Once a tuple is created, it cannot be altered. Tuples are defined by parentheses (), with items separated by commas

Dictionaries

A dictionary in Python is an unordered collection of data values, used to store data values like a map. The dictionaries hold key-value pairs. Keys in a dictionary must be unique and immutable, which typically are strings, numbers, or tuples.

Creating a Dictionary

Dictionaries are defined by (curly) braces { } with each item being a pair in the form key: value.


'''
my_dict = {
    key1: value1,
    key2: value2,
    key3: value3
}
'''

person = {"name": "Alice", "age": 25, "city": "Tainan"}
print(person)

# Simple dictionary
student = {
    "name": "Alice",
    "age": 20,
    "grade": "A"
}

# Dictionary with mixed value types
person = {
    "name": "Bob",
    "age": 25,
    "is_student": False,
    "hobbies": ["reading", "gaming"]
}

# Empty dictionary
empty_dict = {}

Accessing Dictionary Elements

we can access the values in the dictionary by referring to its key.


print(person["name"]) # Outputs 'Alice'

student = {"name": "Alice", "age": 20}
print(student["name"])  # Output: Alice

# Using .get() to avoid KeyError
print(student.get("grade"))  # Output: None
print(student.get("grade", "Not Found"))  # Output: Not Found

Modifying Dictionaries

Dictionaries are mutable. we can add new key-value pairs, modify values, or delete key-value pairs.

Sets

A set is an unordered , mutable collection of unique elements. Sets are useful for storing items where duplicates are not allowed , and order does not matter . Sets are are ideal for performing mathematical set operations like unions, intersections, and differences.

Creating a Set

Sets are defined by curly braces {} or the set() function, and they automatically remove duplicate items.


# my_set = {item1, item2, item3}
colors = {"red", "blue", "green", "red"}
print(colors) # Outputs {"red", "blue", "green"}

# Set of numbers
numbers = {1, 2, 3, 4}

# Set of strings
fruits = {"apple", "banana", "cherry"}

# Mixed data types
mixed = {1, "hello", True}

# Duplicate values are automatically removed
dup_set = {1, 2, 2, 3}
print(dup_set)  # Output: {1, 2, 3}

Set Operations:

Since sets are unordered, you cannot access elements by index. We can loop through the set items, or check if a value is present


# Adding Elements
fruits = {"apple", "banana"}
fruits.add("cherry")
print(fruits)  # Output: {'banana', 'cherry', 'apple'}

# Removing Elements
fruits = {"apple", "banana", "cherry"}
fruits.remove("banana")  # Raises KeyError if element not found
print(fruits)  # Output: {'cherry', 'apple'}

fruits.discard("banana")  # No error if element not found

# Check Membership
if "blue" in colors:
    print("Blue is in the set")
    
# Length of Set
fruits = {"apple", "banana", "cherry"}
print(len(fruits))  # Output: 3

Modifying Sets

We can add items to sets using the add() method, and multiple items using the update() method. Items can be removed using remove() or discard(). 


colors.add("yellow")
colors.discard("green")
print(colors)

Set Algebra (Mathematical Operations)

Sets are ideal for mathematical operations like unions (|), intersections (&), and differences (-).


primary_colors = {"red", "blue", "yellow"}
secondary_colors = {"green", "orange", "purple"}

all_colors = primary_colors | secondary_colors # Union of sets
common_colors = primary_colors & secondary_colors # Intersection of sets
unique_colors = primary_colors - secondary_colors # Difference of sets

A = {1, 2, 3}
B = {3, 4, 5}

# Union
print(A | B)         # Output: {1, 2, 3, 4, 5}
print(A.union(B))   # Same result

# Intersection
print(A & B)         # Output: {3}
print(A.intersection(B))  # Same result

# Difference
print(A - B)         # Output: {1, 2}
print(A.difference(B))    # Same result

# Symmetric Difference (elements in A or B but not both)
print(A ^ B)         # Output: {1, 2, 4, 5}
print(A.symmetric_difference(B))  # Same result

File Handling

In Python, we can read , write , append , and delete files using built-in functions. The `with` statement automatically closes the file after the block is executed.

Open()

Use the built-in `open()` function: The open() function requires the name of the file and the mode in which the file should be opened, such as 'r' for reading, 'w' for writing, **'a' for appending**, and **'b' for binary mode**. Always remember to close the file after use with `file.close()`.


file = open('example.txt', 'mode')

# Read entire file
with open("example.txt", "r") as file:
    content = file.read()
    print(content)

# Read line by line
with open("example.txt", "r") as file:
    for line in file:
        print(line.strip())  # Remove newline characters

# Read all lines as a list
with open("example.txt", "r") as file:
    lines = file.readlines()
    print(lines)

Files write()

Writing to a file involves opening it in write ('w') or append ('a') mode. If a file is opened in write mode, any existing content is erased. In append mode, new content is added at the end of the file


# **Writing to a File**:  `write()`
# Overwrite file or create if not exists
with open("example.txt", "w") as file:
    file.write("Hello, World!\n")
    file.write("This is a new line.")

file = open('example_write.txt', 'w')
file.write("Hello, Python!\n")
file.write("Writing to a file is easy.\n")
file.close()

# **Appending to a File**
# Add to the end of the file
with open("example.txt", "a") as file:
    file.write("\nAppended line.")
file.close()

Handling Exceptions with `try` and `except`

We can catch and handle exceptions using `try` and `except`. 


'''
try:
    # code that may cause an error
except ExceptionType:
    # handle the error
'''
try:
    num = int(input("Enter a number: "))
    result = 100 / num
    print("Result:", result)
except ValueError:
    print("Please enter a valid number.")
except ZeroDivisionError:
    print("Cannot divide by zero.")
