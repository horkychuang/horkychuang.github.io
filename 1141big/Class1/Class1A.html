Introduction

They allow for decision-making and repeating actions, which are fundamental in creating dynamic and responsive programs.

A. Decision Structures

The program always needs to be executed depending upon a particular condition. Python provides the following statements for implementing the decision structure.

  • if statement
  • if else statement
  • if-elif-else Statement
  • nested if statement

1.1 if statement

If the `condition (Boolean expressions) is true`, statement is executed; otherwise it is `skipped` 


# Check if revenue is negative and convert to absolute value for reporting
revenue = float(input("Enter monthly revenue (can be negative): "))

if revenue < 0:
    revenue = -revenue

print("The absolute value of revenue is", revenue, "USD")

1.2 if-else Statement

The `condition` is evaluated first. If the condition is true, statement1 is executed. If the condition is false, statement 2 is executed. 


# Categorize product ID for inventory management
product_id = int(input('Enter product ID: '))

if product_id % 2 == 0:
    print(f"Product ID {product_id} is even → Category: Electronics")
else:
    print(f"Product ID {product_id} is odd → Category: Apparel")

1.3 if-elif-else Statement

The `if-elif-else` statement is used when you have `multiple conditions` to check. It allows you to test multiple conditions and execute the corresponding block of code for the first true condition encountered. If the conditions in the *if* and all *elif* statements evaluate to false, the `else` block is executed.
elif is an abbreviation of "else if." Again, exactly one branch will be executed. There is no limit of the number of elif statements, but the last branch has to be an else statement. 


# Compare Q1 and Q2 sales
q1_sales = float(input("Enter Q1 sales (in USD): "))
q2_sales = float(input("Enter Q2 sales (in USD): "))

if q1_sales < q2_sales:
    print(f"Q1 sales ({q1_sales}) are lower than Q2 sales ({q2_sales}).")
elif q1_sales > q2_sales:
    print(f"Q1 sales ({q1_sales}) are higher than Q2 sales ({q2_sales}).")
else:
    print(f"Q1 and Q2 sales are equal at {q1_sales}.")

1.4 Nested Decision Structures

One `conditional can also be nested`. Nested `if` statements are used when you need to check `multiple conditions` within another condition 


# Alternative nested structure for sales comparison
q1 = float(input("Enter Q1 sales: "))
q2 = float(input("Enter Q2 sales: "))

if q1 == q2:
    print(f"Q1 and Q2 sales are equal → No change.")
else:
    if q1 < q2:
        print(f"Q1 sales are lower → Consider promotional campaign.")
    else:
        print(f"Q1 outperformed Q2 → Review successful strategies.")

B. Repeating Actions

We often require a set of statements to be executed **a number of times**. This type of program execution is called looping. Python provides the following construct.

  • while loop
  • For loop

1. The while Loop

If we want to execute a piece of code as long as the condition is true, we can use a `while` loop. There are three different kinds of loops:

  • Count-controlled loops: repeating a loop for a certain number of times
  • Condition-controlled loop: will be repeated until a given condition changes
  • Collection-controlled loop: through the elements of a 'collection', which can be an array, list or other ordered sequence


# Print motivational message for team
i = 1
while i <= 5:
    print(" I love analyzing business data!")
    i = i + 1  
    

# Process customer list by index
customers = ["Alice Lin", "Bob Chen", "Charlie Wu", "Diana Liu"]
index = 0
while index < len(customers):
    print("Processing customer:", customers[index])
    index += 1
    
# another
# Process order queue using pop()
pending_orders = ["Order-1001", "Order-1002", "Order-1003"]
while pending_orders:
    current_order = pending_orders.pop()
    print(" Shipped:", current_order)
print(" All orders processed.")

2. For Loop

A for loop is used for iterating over a sequence (such as a list, tuple, dictionary, set, or string). First, we can generate a sequence using `range() function`. The general format range (start, stop, step) takes three arguments. Start and Step are the optional arguments. A start argument is a starting number of the sequence. If start is not specified it starts with 0. The stop argument is ending number of sequence. The step argument is linear difference of numbers. The default value of the step is 1 if not specified.

2.1 Range(start, stop, step)


# Loop through days 0 to 9
for day in range(10):
    print(f"Day {day}", end=' ')
print()

# Analyze days 2 to 9
for day in range(2, 10):
    print(f"Analysis for Day {day}", end=' ')
print()

# Every 3rd month: review point
for month in range(3, 31, 3):
    print(f"Review Month {month}", end=' ')
print()

# Countdown to quarter end
for day in range(5, 0, -1):
    print(f"Closing in {day} days...")
print(" Quarter closed!")

3. For loop + Condition Behavior

break exits the loop prematurely, and continue skips the rest of the current iteration and moves to the next one. 


# Stop processing when reaching VIP client
clients = ["Alice Lin", "Bob Chen", "VIP Client", "David Wang"]
for client in clients:
    if client == "VIP Client":
        break
    print("Processing:", client)

# Skip invalid transaction ID
for trans_id in range(1, 20):
    if trans_id == 12:
        continue
    print(f"Processing Transaction {trans_id}")
    

# Skip even weeks in reporting
week = 0
while week < 10:
    week += 1
    if week % 2 == 0:
        continue
    print(f"Generate report for Week {week}")

# Interactive city data input

while True:
    city = input(prompt)
    if city == 'quit':
        break
    else:
        print(f"Collecting data from {city.title()}...")

4. Do-While Loop Behavior

There is no built-in do-while loop like in some other programming languages. However, you can achieve the same behavior using a `while` loop with a `break` statement. Here's how you can emulate a **do-while** loop in Python 


while True:
    user_input = input("Enter 'exit' to stop: ")
    if user_input == 'exit':
        break

This approach is similar to a `do-while` loop where the loop body is guaranteed to execute at least once before the condition is checked

C. Functions and Packages

They allow us to encapsulate a task or a set of instructions into a self-contained block, which can be executed whenever needed

1. Functions

Functions are named blocks of code designed to do one specific job. When we want to perform a particular task that you’ve defined in a function, you call the function responsible for it

1.1 Defining a Function and Call

A function is defined using the `def keyword`, followed by a function name, parentheses (), and a colon :. The `indented` block of code` following the `: ` is the body of the function 


# Define a function to greet users in a BI dashboard
def welcome_user():
    print("Hello! Welcome to the Sales Analytics Dashboard.")

# Calling the function
welcome_user()

1.2 Function with Arguments


# Greet a specific user (e.g., logged-in analyst)
def welcome_user(username):
    print(f"Hello, {username.title()}! Here is your daily report.")

welcome_user('chen')

1.2.1 Arguments and Parameters

We defined greet_user() to require a value for the variable username. Once we called the function and gave it the information (a person’s name), it printed the right greeting.

  • An argument is a piece of information that’s passed from a function call to a function. The value was assigned to the parameter username

1.2.2 Multiple Parameters

A function definition can have `multiple parameters`, a function call may need multiple arguments. You can pass arguments to your functions in a number of ways. You can use positional arguments, which need to be in `the same order` `the parameters` were written. In other words, the number of arguments in the call must match exactly with the function definition 


# Positional Arguments: Analyze sales by region and product
def report_sales(region, product):
    print(f"\nSales data for {region.upper()} region.")
    print(f"Top product: {product.title()}.")

report_sales('taipei', 'laptop')

# Function to calculate revenue: price × quantity
def calculate_revenue(price, quantity):
    return price * quantity

total = calculate_revenue(299.99, 150)  # Price = $299.99, Qty = 150 units
print(f"Total Revenue: ${total:,.2f}")

Keyword argument
A keyword argument is a name-value pair that we pass to a function. We `directly associate` the name and the value within the argument, so when you pass the argument to the function. In other words, pass arguments by `explicitly specifying the name` of the parameter 


# Keyword Arguments: Clear labeling for financial clarity
def report_sales(region, product):
    print(f"\nSales data for {region.upper()} region.")
    print(f"Top product: {product.title()}.")

report_sales(region='kaohsiung', product='wireless earbuds')

1.2.3 Default Values

We can define a default value> for each parameter. If an argument for a parameter is provided in the function call, Python uses the argument value. If not, it uses the parameter’s default value.

1.3 Return Values

A function doesn’t always have to display its output directly. Instead, it can process some data and then return a value or set of values. The value the function returns is called a return value. 


# Returning a Simple Value: Net profit calculation
def calculate_profit(revenue, cost):
    return revenue - cost

profit = calculate_profit(85000, 62000)
print(f"Net Profit: ${profit:,}")

# Returning Multiple Values: Financial metrics
def financial_metrics(revenue, cost):
    profit = revenue - cost
    margin = (profit / revenue) * 100
    return profit, margin

net_profit, profit_margin = financial_metrics(100000, 70000)
print(f"Profit: ${net_profit:,}, Margin: {profit_margin:.1f}%")

# Return a dictionary: Build a customer profile
def build_customer(first_name, last_name, age=None, segment=None):
    customer = {
        'first': first_name,
        'last': last_name
    }
    if age:
        customer['age'] = age
    if segment:
        customer['segment'] = segment
    return customer

top_client = build_customer('linda', 'huang', age=34, segment='premium')
print(top_client)

1.5 Lambda/ Anonymous

A lambda function can take n number of arguments at a time. But it returns only one argument at a time. they don't have a name. We define the lambda keyword in Python to an unnamed function such as lambda input_arguments : expression 


# Lambda function: Inline calculation (e.g., tax)
(lambda price, tax_rate: price * (1 + tax_rate))(299.99, 0.08)

# Named function alternative
def tsmc_tax(price, tax_rate):
    return price * (1 + tax_rate)

tsmc_tax(299.99, 0.08)

1.5.1 with map()、filter() and reduce()


# `map()` function: Apply transformation to all items 
sales_ntd = [25000, 30000, 28000, 32000]
exchange_rate = 0.032  # NTD to USD
sales_usd = list(map(lambda x: x * exchange_rate, sales_ntd))
print("Sales in USD:", [f"${s:.0f}" for s in sales_usd])

# `filter()` method: Extract high-performing days (sales > $900)
daily_sales = [850, 920, 700, 950, 880, 1000]
high_sales_days = list(filter(lambda x: x > 900, daily_sales))
print("High Sales Days (>$900):", high_sales_days)


# List comprehension with lambda: Pre-calculate square targets
squares = [lambda num = num: num ** 2 for num in range(0, 11)]
print("Square values from 0 to 10:")
for square in squares:
    print(square(), end=" ")
print()

# Lambda with if-else: Risk classification
classify_risk = lambda score: "High Risk" if (score < 50) else "Low Risk"
print("\nRisk Level:", classify_risk(42))  # High Risk
